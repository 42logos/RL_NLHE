

# Project layout
The project is structured as follows:

```
nlhe_refactor/
├── docs/                   # Documentation files
├── nlhe/                   # Main package
│   ├── __init__.py
│   ├── conf/               # Configuration files
│   ├── envs/               # Custom environments
│   ├── models/             # Model definitions
│   ├── policies/           # Policy definitions
│   └── train/              # Training scripts
└── tests/                  # Unit tests
```
